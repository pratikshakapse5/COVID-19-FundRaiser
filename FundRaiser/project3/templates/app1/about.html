Welcome to the Fun Raiser

What is this project about?

This Web Project1 is full stack Django based project for Covid-19 funding. Considering need of funcding, This FundRaiser website has Make donations feature. A user can view dashboard and donate his part to the needed funds. Following are the tabs -
  1. Dashboard
    • This is a Dashboard for all the statistics User needs to know. It shows pie chart for Needed funds and Donated funds
    • It gives how many donations are made so far
    • How many donors have contributed to the funding
  2. Make Donations
    • Donor can make a wish why he is interested in donating
  3. My Donations
    • User can view his donations in the past
  4. Make Payment
    • This allows user to make a payment by giving card details

Tools used

  1. Environment- Anaconda
  2. Editor- Microsoft Visual Studio
  3. Websites/resources - Stackoverflow is the greatest resource, Django documentation
  4. Chartist for creating Pie charts

What steps did I follow in order to complete this project?

  1. I have a github account and first thing I did before even starting the project was to create a git repository
  2. Created a Django project in Anaconda environment.
  3. As there was no requirements given, I started with designing for users who raise funds
  4. I faced few problems creating two kinds of uses ones who raise funds and others who donates funds
  5. Then I took other scenario where user can donate money and see statistics and view his own contributions
  6. once user decides which organization he wants to donate to he go ahead with a payment part
  7. once all the modules are done I started with Dashboard.
